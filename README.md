# 수학 기호 사전  
: 2018년도 'KB 국민은행 소프트웨어 경진대회' 참가작입니다.

## Service 개요
<수학기호 사전>은 공부하다 모르는 수학 기호를 Canvas기능을 통해 직접 그리면, 수학기호가 무엇인지, 쓰임이 어떤한지 알려주는 Application이다.


<br></br>

## 기능 
- Machine Learning을 통한 수학기호 분류   
        1. 이미지 인식 Algorithm 사용
